*DOM
(Document Object Model), é um conjunto de objetos dentro do navegador que dá acesso a componentes internos do website.

-Árvore DOM 
estrutura hierárquica do site, começa por "window"

-Seleção de Elementos
"InnerText" utiliza apenas o texto
"InnerHTML" utiliza o texto com suas tags e formatações
Seleção pode ser feita utilizando variáveis ou pelo comando escrito diretamente

-Por Marca
getElementsByTagName()[] Possibilita seleção de 1 ou mais objetos com a mesma tag

-Por ID
getElementsByID() seleciona um elemento pela sua id específica

-Por Nome
getElementsByName()

-Por Classe
getElementsByClassName()

-Por Seletor
querySelector()
querySelecctorAll()
utiiza a sintaxe do css
id = "#"
class = "."
para selecionar um elemento chamado "div id=msg" se escreve querySelector('div#msg'\)

-Eventos DOM
-Exemplos de Eventos Simples
mouseenter- quando o mouse encosta em um determinado elemento
mousemove- quando o mouse se move em cima de um determinado elemento
mousedown- segurar o click em cima do elemento
mouseup- quando solta o botão
click- click normal no elemento
mouseout- quando o mouse sai de cima do elemento

-Funções
função é um conjunto de código executado quando ocorre um determinado evento
estrutura básica:
function ação(parâmetro) {
    bloco (código)
}

a ação pode ser inserida no HTML ou diretamentre no script
HTML: nome da função na tag do elemento desejado.  <div id="area"> onclick="clicar()"
Script:  variável.addEventListener('click', clicar)
