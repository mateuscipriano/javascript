*Variáveis
" = " significa recebe
-Estrutura: var ou let identificador = valor

-Identificadores: podem começar com letra, $ ou _, não podem começar com números, podem ter acentos ou símbolos, sem espaços e não podem ser palavras reservadas como "var".

"typeof" é utilizado para identificar o tipo primitivo ou DataType de uma variável
Exemplo:
> var n = 200                                       Tipos primitivos ou DataTypes:
> typeof n                                          -number (infinity, NaN 'NotaNumber')
'number'                                            -string (cadeia de caracteres)
> n = "aaa"                                         -boolean (true ou false)
'aaa'                                               -null  
> typeof n                                          -undefined
'string'                                            -object (array)
> n = "123"                                         -function
'123'
> typeof n
'string'
 n = false
false
> typeof n
'boolean'

Tratamento de Dados

 fará com que a variável receba o valor resultante do comando descrito (window.prompt envia um "string")

 - " + " tem função de concatenação (conectar) ou adição
 (number + number) "+" terá a função de adição
 (string + string) "+" terá a função de concatenação

 -conversão string to number 
 number.parseInt(n) (número inteiro)
 number.parseFloat(n) (número real (com vírgula))
 number (o js saberá automaticamente se é número real ou inteiro)

 -conversão number to string
 String(n)
 n.toString()

 -Formatação de Strings ou Template Strings
 "${}" é placeholder

 -É possível substituir concatenação: 'Seu nom é' + nome
 por template strings: `Seu nome é ${nome}` (crease é obrigatória)

 -Atributos
 nome.length diz qunatos caracteres a string tem
 nome.toUpperCase() transforma a string em letras maiúsculas
 nome.toLowerCase() transforma a string em letras minúsculas

n1 = 10.5
n1.toFixed(n) muda o número de casas decimais que serão mostradas
.replace ('.', ',') fará com que a tela mostre o número com vírgula ao invés de um ponto
