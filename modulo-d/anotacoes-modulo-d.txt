*Sequências
-if (condição) 2 possibilidades delimitadas por blocos, se for true executa o comando,
-else se não for true, executa o comando alternativo

-Condição Simples
if(condição){
    true
}
caso não seja true, nada de diferente acontece

-Condição Composta
if(condição) {
    true
}
else {
    false
}

-Condição Aninhada
uma condição dentro de outra
if(cond1){
    bloco1
}
  else if (cond2){
    bloco2
  } else {
    bloco3
  }

-Condição Múltipla
utilizada principalmente para valores fixos, um valor específico executa um bloco
switch(expressão){
case valor 1:

break(obrigatório)
case valor 2:

break(obrigatório)
case valor 3:

default(caso nenhum valor acima seja satisfeito):

break(obrigatório)
}
